# Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

<b>Example 1:</b>
```
> <host>127.0.0.1</host>
{"host":"127.0.0.1"}
```
</pre>

<b>Example 2:</b>
```
> {"jdk" : "1.8.9"}
<jdk>1.8.9</jdk>
```

<b>Example 3:</b>
```
> <success/>
{"success": null}
```

<b>Example 4:</b>
```
> {"storage" : null}
<storage/>
```
