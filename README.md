# Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

<b>Example 1:</b>
<pre>
> <host>127.0.0.1</host>
{"host":"127.0.0.1"}
</pre>

<b>Example 2:</b>
<pre>
> {"jdk" : "1.8.9"}
<jdk>1.8.9</jdk>
</pre>

<b>Example 3:</b>
</pre>
> <success/>
{"success": null}
</pre>

<b>Example 4:</b>
<pre>
> {"storage" : null}
<storage/>
</pre>
